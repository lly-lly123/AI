# 如何获取Supabase数据库连接字符串

## 📋 详细步骤

### 方法1：从数据库设置页面获取（推荐）

1. **访问数据库设置页面**
   - 网址：https://supabase.com/dashboard/project/pigeonai/settings/database
   - 或点击左侧边栏：**Settings** → **Database**

2. **找到连接字符串**
   - 在页面中找到 **"Connection string"**（连接字符串）部分
   - 通常位于页面中间或下方
   - 会显示多个标签：**URI**、**JDBC**、**Golang** 等

3. **选择URI标签**
   - 点击 **"URI"** 标签
   - 会显示类似这样的连接字符串：
     ```
     postgresql://postgres:[YOUR-PASSWORD]@db.xxxxx.supabase.co:5432/postgres
     ```

4. **复制连接字符串**
   - 点击连接字符串旁边的复制按钮（📋图标）
   - 或手动选中并复制

5. **替换密码**
   - 连接字符串中的 `[YOUR-PASSWORD]` 需要替换为您创建项目时设置的数据库密码
   - 例如：`postgresql://postgres:your_actual_password@db.xxxxx.supabase.co:5432/postgres`

### 方法2：从项目概览页面获取

1. **访问项目页面**
   - 网址：https://supabase.com/dashboard/project/pigeonai
   - 或点击左侧边栏的 **"Project Settings"**

2. **查看连接信息**
   - 在项目概览页面，通常会显示数据库连接信息
   - 点击 **"Database"** 或 **"Connection Info"** 查看

### 如果找不到连接字符串

如果页面中没有显示连接字符串，可以：

1. **检查项目状态**
   - 确保项目已完全创建完成
   - 项目状态应该显示为 "Active" 或 "Running"

2. **使用API设置页面**
   - 访问：https://supabase.com/dashboard/project/pigeonai/settings/api
   - 获取 **Project URL** 和 **Database Password**
   - 手动构建连接字符串：
     ```
     postgresql://postgres:[数据库密码]@db.[项目引用].supabase.co:5432/postgres
     ```
   - 其中 `[项目引用]` 可以从 Project URL 中提取
   - 例如：如果 Project URL 是 `https://abcdefgh.supabase.co`
   - 那么项目引用就是 `abcdefgh`
   - 连接字符串就是：`postgresql://postgres:[密码]@db.abcdefgh.supabase.co:5432/postgres`

## 🔍 连接字符串格式说明

标准格式：
```
postgresql://postgres:[密码]@db.[项目引用].supabase.co:5432/postgres
```

组成部分：
- `postgresql://` - 协议
- `postgres` - 用户名（固定）
- `[密码]` - 数据库密码（创建项目时设置）
- `db.[项目引用].supabase.co` - 数据库主机
- `5432` - 端口（PostgreSQL默认端口）
- `postgres` - 数据库名（固定）

## ⚠️ 注意事项

1. **密码安全**：连接字符串包含密码，请妥善保管
2. **密码格式**：如果密码包含特殊字符，需要进行URL编码
3. **项目引用**：可以从Project URL中提取，格式为 `https://[项目引用].supabase.co`
