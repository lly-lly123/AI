# 🌐 如何查找您的网站访问地址

## 📍 在Zeabur控制台查找

### 方法1：在项目概览页面查看

1. **访问Zeabur控制台**
   - 打开 https://zeabur.com
   - 登录您的账号

2. **进入您的项目**
   - 点击项目 "untitled-2"
   - 进入项目概览页面

3. **查找访问地址**
   - 在服务卡片上，会显示一个访问地址
   - 通常格式为：`https://untitled-2-xxxxx.zeabur.app`
   - 或者：`https://ai-xxxxx.zeabur.app`
   - 点击地址可以直接访问

### 方法2：在服务详情页面查看

1. **进入服务详情**
   - 在项目页面，点击您的服务（GitHub服务）
   - 进入服务详情页面

2. **查看域名信息**
   - 在服务详情页面的顶部或侧边栏
   - 找到 "Domains" 或 "域名" 部分
   - 会显示分配的域名地址

3. **或者查看Overview**
   - 在服务详情页面，点击 "Overview" 标签
   - 查找 "Add Domain" 卡片
   - 会显示当前使用的域名

### 方法3：查看部署日志

1. **进入部署日志**
   - 在服务页面，点击 "Logs" 或 "日志" 标签
   - 查看最新的部署日志

2. **查找域名信息**
   - 部署完成后，日志中通常会显示访问地址
   - 格式类似：`Deployed to: https://xxxxx.zeabur.app`

## 🔍 网站访问地址格式

Zeabur分配的域名通常有以下格式：

- `https://untitled-2-xxxxx.zeabur.app`
- `https://ai-xxxxx.zeabur.app`
- `https://your-service-name.zeabur.app`

其中 `xxxxx` 是Zeabur自动生成的随机字符串。

## 📋 完整的访问地址

部署成功后，您可以使用以下地址：

### 1. 网站首页
```
https://your-service-name.zeabur.app
```

### 2. 后台管理系统
```
https://your-service-name.zeabur.app/admin.html
```

### 3. API健康检查
```
https://your-service-name.zeabur.app/api/health
```

## ⚠️ 如果找不到访问地址

### 情况1：部署还在进行中

如果服务状态显示 "Deploying" 或 "Building"：
- 等待部署完成（通常3-5分钟）
- 部署完成后会自动显示访问地址

### 情况2：部署失败

如果服务状态显示 "Failed" 或 "Error"：
1. 点击服务查看错误日志
2. 检查环境变量配置
3. 查看部署日志中的错误信息
4. 修复问题后重新部署

### 情况3：需要手动分配域名

如果看不到访问地址：
1. 在服务详情页面，点击 "Domains" 或 "域名"
2. 点击 "Add Domain" 或 "添加域名"
3. Zeabur会自动分配一个域名

## 📱 移动端访问

网站支持移动端访问，会自动跳转到移动版页面：
- 移动端会自动访问：`https://your-service-name.zeabur.app/mobile.html`
