## 🚀 现在只需要点击！

### 步骤：

1. **在Vercel页面上**，找到 "PigeonAI · 1h ago" 这一行

2. **在这一行的右边**，有一个 **"Import"** 按钮（紫色或蓝色的按钮）

3. **点击 "Import" 按钮**

4. **然后会进入项目配置页面**：
   - **Project Name**: 保持默认 `PigeonAI`
   - **Framework Preset**: 会自动检测，保持默认即可
   - **Root Directory**: 保持默认 `./`
   - **Build Command**: 留空（如果显示npm run build，可以删除）
   - **Output Directory**: 留空
   - **Install Command**: 可以填写 `cd backend && npm install`（可选）

5. **点击页面底部的 "Deploy" 按钮**（绿色的大按钮）

6. **等待1-2分钟**，部署完成！

7. **完成后会显示访问地址**，类似：`https://pigeonai-xxxxx.vercel.app`
