## 📋 立即可执行的步骤（不需要GitHub）

### 第1步：注册TeraBox（1TB永久免费）✨

1. **访问**：https://www.terabox.com/
2. **注册账号**：
   - 使用邮箱或手机号注册
   - 验证账号
3. **自动获得1TB空间**：
   - 注册成功后立即获得1TB免费空间
   - 永久免费，无需信用卡
4. **保存登录信息**：
   - 记住账号和密码
   - 稍后需要配置API（如果需要）

**预计时间**：5分钟

---

### 第2步：注册Cloudflare R2（10GB永久免费）✨

1. **访问**：https://dash.cloudflare.com/
2. **注册账号**：
   - 使用邮箱注册（免费）
   - 不需要信用卡
3. **创建R2存储桶**：
   - 登录后，点击左侧菜单 **"R2"**
   - 点击 **"Create bucket"**
   - 名称：`pigeonai`（或您喜欢的名称）
   - 点击 **"Create bucket"**
4. **获取API凭证**：
   - 点击右上角 **"Manage R2 API Tokens"**
   - 点击 **"Create API Token"**
   - 权限选择：**"Admin Read & Write"**
   - 点击 **"Create API Token"**
   - **重要**：记录以下信息：
     - `Account ID`：类似 `abc123def456`
     - `Access Key ID`：类似 `abc123...`
     - `Secret Access Key`：类似 `def456...`（只显示一次！）

**预计时间**：10分钟

---

### 第3步：配置Supabase（1GB永久免费）✨

1. **访问**：https://supabase.com
2. **注册账号**：
   - 使用GitHub账号登录（推荐）
3. **创建项目**：
   - 点击 **"New Project"**
   - 项目名称：`pigeonai`
   - 数据库密码：设置强密码（记住它！）
   - 区域：选择 **"Southeast Asia (Singapore)"** 或 **"East Asia (Tokyo)"**
   - 点击 **"Create new project"**
   - 等待2-3分钟创建完成
4. **获取配置信息**：
   - 点击左侧 **"Settings"** → **"API"**
   - 记录以下信息：
     - `Project URL`：类似 `https://xxxxx.supabase.co`
     - `anon public` key：很长的字符串
5. **创建存储Bucket**（可选）：
   - 点击左侧 **"Storage"**
   - 点击 **"New bucket"**
   - 名称：`files`
   - 权限：选择 **"Public"**（如果是公开文件）

**预计时间**：10分钟

---

### 第4步：准备环境变量配置清单 📝

将所有配置信息整理到一个文档中，方便稍后在Zeabur配置：

```env
# ============================================
# Cloudflare R2配置（10GB永久免费）
# ============================================
CLOUDFLARE_R2_ACCOUNT_ID=你的AccountID
CLOUDFLARE_R2_ACCESS_KEY_ID=你的AccessKey
CLOUDFLARE_R2_SECRET_ACCESS_KEY=你的SecretKey（重要！只显示一次）
CLOUDFLARE_R2_BUCKET_NAME=pigeonai

# ============================================
# Supabase配置（数据库1GB）
# ============================================
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_ANON_KEY=your-anon-key

# ============================================
# TeraBox配置（1TB永久免费）
# ============================================
# 注意：TeraBox可能需要申请API访问权限
TERABOX_ACCESS_TOKEN=（待获取）
TERABOX_REFRESH_TOKEN=（待获取）

# ============================================
# MinIO配置（开源存储，稍后在Zeabur部署）
# ============================================
# 需要在Zeabur部署MinIO服务后配置
MINIO_ENDPOINT=（待部署后填写）
MINIO_ACCESS_KEY=minioadmin
MINIO_SECRET_KEY=（设置强密码）
MINIO_BUCKET=pigeonai
```

**预计时间**：5分钟整理
