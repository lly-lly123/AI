- ✅ 响应式布局适配

## 🌐 获取公网URL的3种方式

### 方式一：Vercel部署（最快，推荐）

1. **访问 Vercel**
   ```
   https://vercel.com
   ```

2. **登录并导入项目**
   - 点击 "New Project"
   - 选择你的GitHub仓库（或直接拖拽项目文件夹）
   - 项目名称：`pigeonai`（或自定义）

3. **部署设置**
   - Framework Preset: `Other`
   - Root Directory: `./`
   - Build Command: **留空**
   - Output Directory: `.`

4. **环境变量（可选，用于AI功能）**
   - 在项目设置中添加：
     - `ZHIPU_API_KEY_EVO` = 你的智谱API Key
     - `ZHIPU_API_KEY_ADMIN` = 你的智谱API Key（管家）

5. **部署**
   - 点击 "Deploy"
   - 等待部署完成（约1-2分钟）

6. **获取公网URL**
   部署完成后，你会得到类似这样的URL：
   ```
   https://pigeonai.vercel.app
   ```

7. **访问统一预览入口**
   ```
   https://pigeonai.vercel.app/preview.html
   ```
   或直接访问根目录（会自动跳转到index.html，然后可以手动访问preview.html）

---

### 方式二：Netlify部署（简单拖拽）

1. **访问 Netlify**
   ```
   https://app.netlify.com
   ```

2. **拖拽部署**
   - 登录后，直接将项目文件夹拖拽到部署区域
   - 或连接GitHub仓库

3. **获取URL**
   ```
   https://your-project-name.netlify.app/preview.html
   ```

---

### 方式三：GitHub Pages（免费）

1. **推送到GitHub**
   ```bash
   git add .
   git commit -m "添加统一预览入口"
   git push origin main
   ```

2. **启用Pages**
   - 进入仓库 Settings
   - 找到 Pages 选项
   - Source: 选择 `main` 分支
   - Folder: 选择 `/ (root)`

3. **获取URL**
   ```
   https://your-username.github.io/your-repo-name/preview.html
   ```
