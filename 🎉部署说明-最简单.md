## 🚀 方法1：使用命令行部署（最简单，推荐）

### 执行这个命令：

在终端执行：

```bash
cd "/Users/macbookair/Desktop/智慧鸽系统备份文件/智鸽系统_副本"
npx --yes vercel --prod
```

### 步骤：

1. **执行命令后**，会打开浏览器
2. **登录Vercel账号**（如果没有，使用GitHub账号快速注册）
3. **授权访问**（点击"Authorize"）
4. **回到终端**，会提示一些问题：
   - `Set up and deploy?` → 输入 `y` 或直接回车
   - `Which scope?` → 选择你的账户
   - `Link to existing project?` → 输入 `n`（创建新项目）
   - `What's your project's name?` → 直接回车（使用默认名称）
   - `In which directory is your code located?` → 直接回车（当前目录）

5. **等待部署**（1-2分钟）

6. **完成！** 会显示访问地址，类似：
   ```
   🔗  https://pigeonai-xxxxx.vercel.app
   ```

## 🌐 方法2：网页上传部署（如果你不想用命令行）

### 步骤：

1. **访问**：https://vercel.com
2. **登录**（使用GitHub账号）
3. **点击** "Add New..." → "Project"
4. **找到上传选项**：
   - 如果有"Upload"按钮 → 点击
   - 如果没有，可能需要先连接GitHub，但你可以：
     - 创建一个空的GitHub仓库
     - 或者使用CLI方式（方法1）

5. **上传** `pigeonai.zip` 文件

6. **点击部署**
