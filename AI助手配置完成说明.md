## 🚀 现在您需要做什么

### 步骤1：获取API Key（5分钟）

**推荐：智谱AI**（最简单，国内可访问）

1. 打开浏览器访问：**https://open.bigmodel.cn/**
2. 注册/登录账号
3. 进入"控制台" → "API管理"
4. 创建并复制API Key

> 📖 详细步骤见：`获取API Key详细指南.md`

---

### 步骤2：配置API Key（1分钟）

**方法A：使用配置脚本（推荐）**

```bash
cd 智鸽系统_副本/backend
node setup-ai.js
```

按提示操作即可！

**方法B：手动配置**

1. 创建/编辑文件：`智鸽系统_副本/backend/.env`
2. 添加：
   ```env
   ZHIPU_API_KEY=你的API_Key_粘贴在这里
   ```
3. 保存

---

### 步骤3：测试配置（30秒）

```bash
cd 智鸽系统_副本/backend
node test-ai.js
```

如果看到"✅ AI服务响应成功！"，说明配置正确！

---

### 步骤4：重启后端服务

如果后端服务正在运行：

```bash
# 停止当前服务（按 Ctrl+C）
# 然后重新启动
cd 智鸽系统_副本/backend
npm start
```

## ❓ 常见问题

### Q: 必须配置API Key吗？
A: 不是必须的。如果不配置，系统会使用改进的本地AI逻辑，但功能有限。建议配置以获得更好的AI体验。

### Q: 哪个AI服务最好？
A: 推荐智谱AI，因为：
- 国内可访问，无需VPN
- 注册简单
- 有免费额度
- 响应速度快

### Q: 配置后还是不工作？
A: 
1. 检查 `.env` 文件位置是否正确（应在 `backend` 目录下）
2. 检查API Key格式（没有多余空格）
3. 重启后端服务
4. 运行 `node test-ai.js` 查看详细错误

### Q: 如何查看日志？
A: 查看 `backend/logs/app.log` 文件
