## 🔧 使用说明

### 1. 启动智能中枢管家

```bash
cd backend/core-admin
npm start
```

### 2. 查看系统状态

访问 `http://localhost:3001/api/status` 查看完整的系统状态，包括：
- 核心功能状态
- 增强功能状态
- **智能功能状态**（新增）

### 3. 获取智能建议

```bash
# 获取智能决策建议
curl http://localhost:3001/api/intelligent/advice?category=systemHealth

# 获取预测性维护信息
curl http://localhost:3001/api/intelligent/predictions
```

### 4. 管理工作流

```bash
# 查看所有工作流
curl http://localhost:3001/api/intelligent/workflows

# 手动执行工作流
curl -X POST http://localhost:3001/api/intelligent/workflows/{workflowId}/execute
```

### 5. 查看告警

```bash
# 查看所有告警
curl http://localhost:3001/api/intelligent/alerts

# 查看活跃告警
curl http://localhost:3001/api/intelligent/alerts?status=active
```

### 6. 生成智能报告

```bash
# 生成每日报告
curl -X POST http://localhost:3001/api/intelligent/reports/generate \
  -H "Content-Type: application/json" \
  -d '{"type": "daily"}'

# 查看最新报告
curl http://localhost:3001/api/intelligent/reports?type=daily
```
