## 🚀 最简单的方法（2步完成）

### 第1步：获取GitHub Token（如果还没有）

1. **访问**：https://github.com/settings/tokens
2. **点击** "Generate new token" → "Generate new token (classic)"
3. **填写**：
   - Note: `推送代码`
   - Expiration: `90 days` 或 `No expiration`
   - **勾选** `repo` 权限
4. **点击** "Generate token"
5. **复制Token**（以`ghp_`开头，只显示一次！）

---

### 第2步：执行推送脚本

我已经为你创建了一个自动推送脚本。

#### 在Finder中找到：

1. **打开Finder**
2. **进入项目文件夹**：`智慧鸽系统备份文件/智鸽系统_副本`
3. **找到文件**：`自动推送.sh`
4. **双击运行**（或右键 → "打开方式" → "终端"）

#### 或者直接告诉终端位置：

**我会帮你执行这个脚本**，你只需要在提示时输入Token即可。
