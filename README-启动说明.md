# 🚀 一键启动预览脚本使用说明

## 📋 功能说明

这个脚本可以一键启动本地服务器并自动打开浏览器，模拟真实网络访问场景，方便你验证网站效果。

## 🎯 使用方法

### 方法一：双击运行（推荐）

1. 在 Finder 中找到 `启动预览.command` 文件
2. **双击运行**即可
3. 脚本会自动：
   - 启动本地服务器（端口8000）
   - 打开浏览器访问所有页面
   - 显示服务器运行信息

### 方法二：终端运行

```bash
cd /Users/macbookair/Desktop/AI
./一键启动预览.sh
```

### 方法三：使用已部署的远程URL

编辑 `一键启动预览.sh` 文件，修改 `REMOTE_URL` 变量：

```bash
REMOTE_URL="https://your-domain.vercel.app"
```

然后运行脚本，它会直接打开远程URL，不启动本地服务器。

## 📱 自动打开的页面

脚本会自动打开以下页面：

1. **PC端主站** - `index.html`
2. **移动端** - `mobile.html`
3. **iPhone 15 Pro Max预览** - `iPhone15预览.html`
4. **智能管理系统** - `admin.html`

## ⚙️ 配置选项

在 `一键启动预览.sh` 中可以修改：

- `PORT=8000` - 本地服务器端口
- `REMOTE_URL=""` - 远程部署URL（如果已部署）

## 🛑 停止服务器

- 在终端中按 `Ctrl+C` 停止服务器
- 脚本会自动清理后台进程

## 🔧 故障排除

### 问题1：端口被占用

如果端口8000已被占用，脚本会尝试使用现有服务器。或者修改脚本中的 `PORT` 变量。

### 问题2：Python未安装

macOS通常自带Python3，如果没有，可以安装：
```bash
brew install python3
```

### 问题3：无法打开浏览器

脚本会自动尝试打开浏览器，如果失败，请手动访问显示的URL。

## 💡 提示

- 修改代码后，只需刷新浏览器即可看到更新
- 本地服务器会持续运行，直到你按 Ctrl+C 停止
- 所有页面会在新标签页中打开

## 📞 技术支持

如有问题，请检查：
1. Python是否已安装
2. 端口是否被占用
3. 文件权限是否正确（已自动设置）
