# 获取AI API Key详细指南

## 🎯 推荐方案：智谱AI（最简单，国内可访问）

### 步骤1：注册账号
1. 打开浏览器，访问：**https://open.bigmodel.cn/**
2. 点击右上角"注册"或"登录"
3. 使用手机号或邮箱注册账号
4. 完成邮箱/手机验证

### 步骤2：获取API Key
1. 登录后，进入"控制台"或"API管理"
2. 找到"API Key"或"密钥管理"
3. 点击"创建API Key"或"新建密钥"
4. 复制生成的API Key（格式类似：`abc123def456...`）
5. **重要**：API Key只显示一次，请立即保存！

### 步骤3：配置到系统
将获取的API Key配置到系统中（见下方配置步骤）

---

## 🔄 备选方案：通义千问（阿里云）

### 步骤1：注册账号
1. 访问：**https://dashscope.aliyun.com/**
2. 使用阿里云账号登录（如果没有，先注册阿里云账号）
3. 开通"通义千问"服务

### 步骤2：获取API Key
1. 进入"API密钥管理"
2. 创建新的API Key
3. 复制API Key

### 步骤3：配置到系统
将API Key配置到系统中

---

## ⚙️ 配置步骤

### 方法1：使用配置脚本（推荐）

运行配置脚本，按提示输入API Key：

```bash
cd 智鸽系统_副本/backend
node setup-ai.js
```

### 方法2：手动配置

1. 在 `智鸽系统_副本/backend` 目录下创建 `.env` 文件（如果不存在）
2. 添加以下内容：

```env
# 智谱AI API Key（推荐）
ZHIPU_API_KEY=你的API_Key_在这里

# 或者使用通义千问
# QWEN_API_KEY=你的API_Key_在这里
```

3. 保存文件
4. 重启后端服务

---

## ✅ 验证配置

配置完成后，运行测试脚本验证：

```bash
cd 智鸽系统_副本/backend
node test-ai.js
```

如果看到"AI服务连接成功"，说明配置正确！

## 💡 常见问题

### Q: 智谱AI免费吗？
A: 是的，智谱AI提供免费额度，足够日常使用。

### Q: 需要VPN吗？
A: 不需要！智谱AI和通义千问都可以在国内直接访问。

### Q: API Key在哪里找？
A: 
- 智谱AI：登录后进入"控制台" → "API管理" → "API Key"
- 通义千问：登录后进入"API密钥管理"

### Q: 配置后还是不工作？
A: 
1. 检查 `.env` 文件是否在正确位置
2. 检查API Key是否正确（没有多余空格）
3. 重启后端服务
4. 查看日志文件 `logs/app.log`
