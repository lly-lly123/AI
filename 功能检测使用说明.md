# 🔍 网站功能自动检测使用说明

## 📋 检测脚本说明

已创建完整的自动检测脚本，可以全面检测网站的各项功能。

## 🚀 快速开始

### 方法一：双击运行（推荐）

1. 在 Finder 中找到 `功能检测.command` 文件
2. **双击运行**即可
3. 脚本会自动检测所有功能

### 方法二：终端运行

```bash
cd /Users/macbookair/Desktop/AI
./网站功能检测.sh
```

### 方法三：检测远程URL

```bash
./网站功能检测.sh https://your-domain.vercel.app
```

```

检测内容：
- API Key配置文件检查

- Evo助手API集成检测

### 完整功能检测

运行所有检测（包括API检测）：

```

```bash
./网站功能检测.sh
```

### 场景2：部署后验证

部署后检测远程URL：

```bash
./网站功能检测.sh https://your-domain.vercel.app
```

## 🔍 常见问题

### Q: 检测显示服务器未运行？
A: 先运行 `./一键启动预览.sh` 启动本地服务器

### Q: 如何检测远程部署的网站？
A: 运行 `./网站功能检测.sh https://your-domain.com`

### Q: API Key检测显示未配置？
A: API Key存储在浏览器localStorage中，需要在浏览器中配置。检测脚本会提示这一点。

### Q: 移动端跳转检测失败？
A: 移动端跳转需要JavaScript执行，检测脚本会模拟但可能无法完全测试。建议在实际浏览器中验证。

## 📝 检测清单

运行检测前，确保：

- [ ] 本地服务器已启动（如需要）
- [ ] 所有文件都在项目目录中
- [ ] 已安装 curl 或 wget（用于网络检测）
- [ ] 已安装 Node.js（用于API详细检测，可选）

检测完成后：

4. 确认无误后部署
