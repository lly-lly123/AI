## 🔧 配置AI服务（如果AI无法回答）

### 选项1：配置智谱AI（推荐）

#### 步骤1：获取API Key

1. 访问：https://open.bigmodel.cn/
2. 注册/登录账号
3. 创建API Key
4. 复制API Key

#### 步骤2：在Zeabur中配置

1. 登录Zeabur控制台
2. 进入您的项目
3. 点击主服务 → "Variable" 标签
4. 添加环境变量：
   ```
   ZHIPU_API_KEY=你的API Key
   ```
5. 保存并重启服务

#### 步骤3：测试

1. 重启服务后
2. 在浏览器中测试Evo助手
3. 应该可以正常回答

## 🐛 常见问题

### Q1: 看不到Evo助手图标？

**解决：**
1. 检查浏览器控制台是否有错误
2. 刷新页面（Cmd+Shift+R）
3. 检查是否已登录

### Q2: AI无法回答？

**解决：**
1. 配置智谱API Key（见上方）
2. 检查后端服务是否运行
3. 查看浏览器控制台错误信息

### Q3: 如何获取认证Token？

**在浏览器中：**
1. 打开开发者工具（F12）
2. 切换到 "Console" 标签
3. 运行：
   ```javascript
   localStorage.getItem('auth_token')
   ```
4. 复制返回的token值
