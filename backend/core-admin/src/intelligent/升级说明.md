# 智能模块升级说明

> 中枢管家智能模块升级说明

## 📦 升级内容

### 新增智能模块

#### 1. 智能决策引擎 (`decision-engine.js`)

**功能**：
- 基于规则和AI的自动决策系统
- 支持4个决策类别
- 自动执行低风险决策
- 决策历史学习

**决策类别**：
- `systemHealth` - 系统健康
- `apiHealth` - API健康
- `dataQuality` - 数据质量
- `userBehavior` - 用户行为

#### 2. 预测性维护模块 (`predictive-maintenance.js`)

**功能**：
- 预测API故障
- 预测数据质量问题
- 预测系统性能问题
- 预测用户行为异常

**预测准确率**：
- API故障：75%
- 数据质量：80%
- 系统性能：70%
- 用户行为：65%

#### 3. 自动化工作流引擎 (`workflow-engine.js`)

**功能**：
- 自动化常见管理任务
- 支持条件执行
- 支持重试机制
- 工作流模板系统

**内置工作流**：
- `dailyHealthCheck` - 每日健康检查
- `dataCleanup` - 数据清理
- `performanceOptimization` - 性能优化
- `securityScan` - 安全扫描
- `intelligentBackup` - 智能备份

#### 4. 智能告警系统 (`alert-system.js`)

**功能**：
- 智能分析和分级告警
- 告警冷却机制
- AI增强分析
- 多渠道通知支持

**告警级别**：
- Critical（关键）
- High（高）
- Medium（中）
- Low（低）

#### 5. 自适应优化模块 (`adaptive-optimizer.js`)

**功能**：
- 根据系统状态自动调整配置
- 性能基线跟踪
- 优化效果验证
- 配置自动合并

#### 6. 知识图谱系统 (`knowledge-graph.js`)

**功能**：
- 构建系统知识库
- 实体关系管理
- 智能查询
- 问题解决方案记录

#### 7. 智能报告生成器 (`report-generator.js`)

**功能**：
- AI生成管理报告
- 多类型报告（每日/每周/每月）
- 多章节结构化报告
- AI摘要生成

## 🔗 模块集成

所有智能模块已集成到：
- `core/admin-agent.js` - 管理员代理
- `core/core-admin-manager.js` - 核心管理器
- `routes/api.js` - API路由

## 📡 API端点

所有智能功能可通过以下API访问：
- `/api/intelligent/advice` - 智能决策建议
- `/api/intelligent/predictions` - 预测性维护
- `/api/intelligent/workflows` - 工作流管理
- `/api/intelligent/alerts` - 告警管理
- `/api/intelligent/optimization` - 优化状态
- `/api/intelligent/knowledge` - 知识图谱查询
- `/api/intelligent/reports` - 智能报告

## 📊 性能指标

- **自动化率**：> 80%
- **智能化率**：> 60%
- **预测准确率**：> 75%
- **效率提升**：问题发现时间缩短90%
