# 🚀 快速配置AI助手 - 3步完成

## 第一步：获取API Key（推荐智谱AI）

### 最简单的方法：

1. **打开浏览器**，访问：**https://open.bigmodel.cn/**

2. **注册账号**（如果还没有）：
   - 点击右上角"注册"
   - 使用手机号或邮箱注册
   - 完成验证

3. **获取API Key**：
   - 登录后，进入"控制台"
   - 找到"API管理"或"密钥管理"
   - 点击"创建API Key"
   - **复制API Key**（类似：`abc123def456...`）

> 💡 **提示**：API Key只显示一次，请立即保存！

---

## 第二步：配置API Key

### 方法A：使用配置脚本（最简单）✨

打开终端，运行：

```bash
cd 智鸽系统_副本/backend
node setup-ai.js
```

按提示操作：
1. 选择"1"（智谱AI）
2. 输入"y"（已获取API Key）
3. 粘贴您的API Key
4. 完成！

### 方法B：手动配置

1. 打开文件：`智鸽系统_副本/backend/.env`（如果不存在，创建它）

2. 添加以下内容：
   ```env
   ZHIPU_API_KEY=你的API_Key_粘贴在这里
   ```

3. 保存文件

---

## 第三步：测试配置

运行测试脚本：

```bash
cd 智鸽系统_副本/backend
node test-ai.js
```

如果看到"✅ AI服务响应成功！"，说明配置正确！

---

## 第四步：重启服务

如果后端服务正在运行，需要重启：

```bash
# 停止当前服务（按 Ctrl+C）
# 然后重新启动
cd 智鸽系统_副本/backend
npm start
```

---

## 完成！🎉

现在打开助手，发送"你好"，应该会收到AI的智能回复了！

## ❓ 遇到问题？

### 问题1：找不到API Key
**解决**：
- 智谱AI：登录后 → 控制台 → API管理 → 创建API Key
- 详细步骤见：`获取API Key详细指南.md`

### 问题2：配置后还是不工作
**解决**：
1. 检查 `.env` 文件是否在 `backend` 目录下
2. 检查API Key是否正确（没有多余空格）
3. 重启后端服务
4. 运行 `node test-ai.js` 查看详细错误

### 问题3：测试脚本报错
**解决**：
- 查看错误信息
- 确认API Key格式正确
- 检查网络连接
- 查看 `logs/app.log` 日志文件
