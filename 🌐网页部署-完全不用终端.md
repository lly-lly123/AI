# 🌐 网页部署 - 完全不用终端（5分钟完成）

## ✅ 我已经为你准备好了所有东西！

项目文件已经准备好，现在只需要在网页上操作即可。

---

## 🚀 步骤1：打开Vercel网站（30秒）

1. **打开浏览器**（Safari或Chrome）
2. **访问**：https://vercel.com
3. **点击右上角** "Sign Up" 或 "Log In"

---

## 🔐 步骤2：登录（1分钟）

### 如果没有Vercel账号：

1. **点击** "Sign Up"
2. **选择** "Continue with GitHub"（用GitHub账号登录，最简单）
3. **授权访问**（点击绿色的"Authorize Vercel"按钮）
4. **完成！** 自动登录

### 如果已有账号：

1. **点击** "Log In"
2. **使用GitHub登录**（推荐）

---

## 📤 步骤3：创建项目并上传（2分钟）

### 方法A：直接上传文件夹（最简单）

1. **登录后**，点击右上角的 **"Add New..."** → **"Project"**

2. **如果看到GitHub仓库列表**：
   - 不用管它，**往下滚动**或**点击"Cancel"**
   - 或者直接访问：https://vercel.com/new

3. **找到上传选项**：
   - 页面上应该有一个区域显示 "Drag & Drop" 或 "Upload"
   - 或者点击 "Import" → "Upload"

4. **上传文件**：
   - **方式1**：直接**拖拽整个项目文件夹** `智鸽系统_副本` 到页面上
   - **方式2**：点击"选择文件"，选择项目文件夹中的所有文件（全选）

5. **等待上传完成**（几秒钟）

---

### 方法B：如果找不到上传选项（使用CLI，但我会帮你）

如果Vercel网页版没有直接上传选项，我可以创建一个自动化脚本，你只需要双击运行即可。

---

## ⚙️ 步骤4：配置项目（30秒）

上传后，Vercel会自动检测配置：

1. **Framework Preset**: 保持默认（Other 或自动检测）
2. **Root Directory**: 保持默认 `./`
3. **Build Command**: **留空**（不需要构建）
4. **Output Directory**: **留空**
5. **Install Command**: 可以填写 `cd backend && npm install`（如果需要安装后端依赖）

**大部分保持默认即可！**

---

## 🚀 步骤5：部署（1分钟）

1. **点击页面底部的** "Deploy" 按钮（绿色的大按钮）

2. **等待部署**（1-2分钟）
   - 会显示部署进度
   - 显示 "Building..." 和 "Deploying..."

3. **部署完成！**
   - 会显示绿色的 "Success" 或 "Deployed"
   - **会显示访问地址**，类似：`https://pigeonai-xxxxx.vercel.app`
