# 📝 部署执行说明

## ✅ 脚本已准备就绪

我已经创建了多个部署脚本，您可以选择任一方式部署：

## 🚀 方法1：直接运行部署脚本（推荐）

在终端运行：

```bash
cd "/Users/macbookair/Desktop/智慧鸽系统备份文件/智鸽系统_副本"
bash 直接部署.sh
```

脚本会自动：
1. ✅ 检查并安装依赖
2. ✅ 部署到Vercel
3. ✅ 显示访问地址

**如果首次使用，会提示登录Vercel：**
- 脚本会自动打开浏览器
- 使用GitHub账号登录Vercel
- 登录完成后返回终端继续部署

## 🚀 方法2：手动执行部署命令

如果脚本无法运行，可以直接执行：

```bash
cd "/Users/macbookair/Desktop/智慧鸽系统备份文件/智鸽系统_副本"

# 1. 安装依赖
cd backend
npm install
cd ..

# 2. 登录Vercel（首次需要）
npx vercel login

# 3. 部署
npx vercel --prod --yes
```

## 🚀 方法3：使用Vercel Web界面（最简单）

1. 访问：https://vercel.com
2. 登录账号
3. 点击 "Add New Project"
4. 选择 "Upload"
5. 选择整个 `智鸽系统_副本` 文件夹
6. 点击 "Deploy"

## 📋 可用的部署脚本

- `快速部署.sh` - 快速部署版本

## 🌐 部署后访问

部署完成后，您将获得类似这样的地址：
- 主站：`https://pigeonai-xxxxx.vercel.app`
- 后台：`https://pigeonai-xxxxx.vercel.app/admin.html`
- API：`https://pigeonai-xxxxx.vercel.app/api/health`
