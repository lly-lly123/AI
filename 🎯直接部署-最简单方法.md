# 🎯 直接部署 - 最简单方法（5分钟完成）

## ✅ 不用命令行！直接在网页上部署！

---

## 🚀 步骤1：准备项目文件（打包）

### 选项A：使用我创建的脚本（推荐）

在终端执行：

```bash
cd "/Users/macbookair/Desktop/智慧鸽系统备份文件/智鸽系统_副本"
zip -r pigeonai.zip . -x "*.git*" -x "*node_modules*" -x "*logs*" -x "*.log" -x "*data/*.json"
```

这会在当前目录创建一个 `pigeonai.zip` 文件。

### 选项B：手动打包

1. 在Finder中，进入项目文件夹：`智鸽系统_副本`
2. **全选所有文件**（Command + A）
3. **右键点击** → "压缩"项" 
4. 会生成一个 `.zip` 文件

---

## 🌐 步骤2：在Vercel网页部署（3分钟）

### 2.1 访问Vercel

1. **打开浏览器**（Safari/Chrome）
2. **访问**：https://vercel.com
3. **点击右上角** "Sign Up" 或 "Log In"
4. **使用GitHub账号登录**（推荐，最简单）

### 2.2 创建新项目

1. **登录后**，点击右上角的 **"Add New..."** → **"Project"**
2. 如果看到导入GitHub仓库的选项，**不用管它**
3. **在页面上找到** "Import Project" 或 "Deploy" 按钮
4. **选择** "Upload" 或 "Deploy" → "Upload Files"

### 2.3 上传文件

1. **点击** "Upload" 或拖拽文件区域
2. **选择刚才创建的zip文件**（`pigeonai.zip`）
3. 或者**直接拖拽整个项目文件夹**到页面上

### 2.4 配置项目

Vercel会自动检测，保持默认设置即可：

- **Framework Preset**: Other（或自动检测）
- **Root Directory**: `./`（默认）
- **Build Command**: 留空
- **Output Directory**: 留空

### 2.5 部署

1. **点击** "Deploy" 按钮
2. **等待1-2分钟**，部署完成
3. **会显示访问地址**，类似：`https://pigeonai.xxxxx.vercel.app`
